== Description ==
<p>Now accept top 30+ major crypto currencies donation inside your WordPress website by using this free <strong>cryptocurrency donation box</strong>  - crypto widget for WordPress. Simply add your coin wallet address and use this shortcode inside any page/post/Sidebar - <strong>[crypto-donation-box]</strong>.</p>
<pre>Check Plugin Demo - <a href="http://cryptowidget.coolplugins.net/cryptocurrency-donation-box/">Click Here</a></pre>

https://www.youtube.com/watch?v=FH6LZRapS9Q

<h3>Plugin Features</h3>
<ul>
<li>Integrates Hassle free Tip with MetaMask button.</li>
<li>Convert your coin payment address into QR code. (Using secure PHP QR code library)</li>
<li>Supports top 30+ crypto coins - Bitcoin, Ethereum, Bitcoin Cash, Litecoin, Cardano & much more...</li>
<li>Beautiful crypto donation box design.</li>
<li>[crypto-donation-box] - Add shortcode anywhere page/post/Sidebar.</li>
<li>Responsive / Mobile friendly crypto plugin.</li>
<li>Accept more donation with the help of trusted source<strong>MetaMask</strong></li>
</ul>
